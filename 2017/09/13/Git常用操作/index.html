######3.删除本地仓库某个分支
想删除本地仓库某个分支，执行：git branch -D <local_branch_name>,例如：想删除本地仓库widget2分支，执行命令：git branch -D widget2；

######4.删除远端仓库某分支
删除远端仓库分支和删除本地类似：git push origin :{remote_branch_name}，例如删除远端develop分支：git push origin :develop；
